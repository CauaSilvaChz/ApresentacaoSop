Elementos do Hardware

O que é HARDWARE:

O hardware são peças físicas que compõem um computador. Alguns exemplos são: a placa mãe, monitor, o teclado e o disco rígido. O hardware pode ser dividido nos seguintes grupos:

Dispositivos de entrada:

Um dispositivo de entrada são peças que são conectadas externamente no computador, com a ideia de entrar com dados. Alguns bons exemplos são o mouse e o teclado

Dispositivos de saida:

Um dispositivo de saída é um dispositivo que exibem algo para o usuário após a entrada, como um monitor ou uma impressora

Componentes internos:

Os componentes internos são as peças de hardware que são conectadas entre si para fazer um computador funcionar, como a placa-mãe, o processador e a placa de vídeo, alem de outras peças que falaremos a seguir:

Placa-mãe:

É a placa central de todo computador, onde todos os outros componentes são conectados.

Processador:

O processador (CPU) fica acoplado à placa-mãe. Ele é responsavel por fazer o controle das operações que a máquina realiza. Ele também é conhecido como o cérebro do computador.

Placa de video:

Componente reposnsavel por permitir a visualização de imagens no monitor.

Memoria RAM:

É a peça responsavel por armazenar momentaneamente os dados dos programas que estão em execução no computador, ou seja, enquanto o computador estiver ligado

Hard Disk:

Conhecido como HD, é o equipamento que armazena os dados permanentes no computador, como os documentos de texto e imagens salvas pelo usuário

Perifericos:

Os periféricos são dispositivos ligados ao computador para fazer o processamento de dados, como os exemplos abaixo:

Monitor:

O monitor gera a imagem transmitida pela placa de video.

PenDrive:

É um dispositivo de armazenamento de dados portátil, que pode ser plugado com entradas USB

Teclado:

Serve para o usuário digitar as informações

Outros...

Mouse, impressora, scanner, webcam, microfone, etc...
